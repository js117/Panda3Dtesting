Heightmap brush

Heightmap

A heightmap produced with the editor

A height map is an 2D grayscale image where the color of every pixel is correlated to the height of a point on the 3D map. The Koparka editor allows to edit a height map in real time and show how a terrain rendered with that map will look. Even though a user is actually drawing a 2d image, that virtual 2d canvas is never shown – only how it looks applied to a 3D mesh.

The Brush

The features drawn on the terrain are controlled with a brush. You can change the position, shape, size, rotation and strength of the brush. Holding the left mouse button will start drawing the selected feature on the terrain.

The position of the brush is controlled with the mouse and displayed as a vertical line with two triangle arrows (blue and red) at one end (called the 3D cursor). The brush is automatically moved to where the mouse cursor points, but not to a point on the terrain mesh but on the grid plane.

In recent versions of the editor you can use a option called gl-picking - without going into to much detail, it is enough to say that using this option the 3D cursor is placed directly on the terrain mesh and not on the grid plane. It has a performance impact and it's disabled by default, to enable it change a config variable called "koparka-gl-select" to True (in the cfg.prc file).

The shape of the brush can be selected by clicking on any of the brush shape buttons located at the top of the window. The currently selected brush is highlighted green.

The size of the brush can be changed with A (smaller) and D (larger) keys. Holding down the A key will make the brush smaller, holding the D key will make it larger. At all times the current size of the brush is displayed as a numerical value in the bottom left corner of the window.

The size of the brush can also be changed, by clicking on the config button, entering a value in the Brush Scale field and clicking the OK button.

The rotation of the brush can be changed using the Q and E buttons. Holding the Q button will rotate the brush anti-clockwise, and holding the E button will rotate it clockwise. At all times the rotation of the brush is displayed as a numerical value on the third icon in the bottom left corner of the window.

Just as the size, the rotation can be changed in the configuration dialog. The field labeled Heading is responsible for the brush rotation.

The strength of the brush can be controlled with W and S keys. Holding down the W key will increase the brush strength, whereas holding down the S key will decrease it. The strength of the brush is actually its transparency, where 0.0 is totally transparent and 1.0 is opaque. Using the Raise and Lower paint modes the strength controls how fast the feature is applied, using the Level mode it sets the desired level (height). At all times the current strength of the brush is displayed as a numerical value on the second icon in the bottom left corner of the window.

In recent versions of the editor you can also change the size, strength and rotation of a brush by clicking the appropriate icon and adjusting a pop-up slider.

A pop-up slider

Like other brush setting it can also be set using the configuration dialog.
