Interface overview

GUI

Screen showing basic UI elements

The user interface consists of a few basic elements. The working area taking up most of the window is a 3D view of the map. The 3D cursor is a black vertical line with two red and blue triangles at one end that shows a point on the map that will be edited or a spot on it where objects will be placed. There are several panels with buttons and some with input fields. From time to time the editor may also show a dialog window with both input fields and buttons. Hovering the cursor over any GUI element will always display a tooltip telling you what function the button holds and what keys are associated with that function.

Camera control

By default the camera is controlled with the mouse. If you hold the right mouse button (or the Alt key) then you can pan the camera by simply moving you mouse cursor. If you hold the middle mouse button (or the Ctrl key) then your cursor movement will be translated into camera rotation.

You can zoom in and out using the mouse wheel or the + and – keys. You can also control the speed of the zoom by holding Alt (slower) or Ctrl (faster) keys.

Brush control

You can change the size and strength of the current brush by typing in values in a configuration dialog window (the icon with the gears in the bottom row), but a much more convenient way to do that is to use the keyboard shortcuts.

You can change the size of the brush using A (smaller) and D (bigger) keys.

You can change the strength of a brush using W (more) and S (less) keys.

You can rotate the brush using Q(left) and E (right) keys.

In recent versions of the editor you can also change the size, strength and rotation of a brush by clicking the appropriate icon and adjusting a pop-up slider.

In modes that use the brush for painting, the shape of the brush can be selected by clicking on one of the icons in the top row. The currently selected brush is highlighted green.

You can ‘paint’ by holding down the left mouse button.

Edit Modes

The editor has a few modes, this allows to minimize the amount of widgets displayed in the window and to always have all the currently needed options at hand. These modes are:
  • Heightmap paint mode
  • Texture paint mode
  • Grass paint mode
  • Walkmap paint mode
  • Object place mode
You can change the mode using the buttons in the bottom row, or using the appropriate hotkey (F1-F5). More details on each mode can be found in the next chapters.
