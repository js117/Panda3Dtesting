Walk map and navigation mesh

Walkmap painting

In Walkmap mode you can mark some parts of the terrain as unwalkable. The resulting map can be useful by itself (it may be cheaper/faster to sample a pixel from an image then to run collision detection), but the editor can generate a navigation mesh – a special graph that is needed for the PandaAI pathfinding system that ships with Panda3D.

In this mode you have two tools, represented by icons in the bottom right corner. The red icon allows to paint areas of the map that are inaccessible and the green icon allows to paint accessible areas. By default all of the map is accessible (walkable).

Areas that are unwalkable are displayed with a red tint (in this mode and in object paint mode).

In the Walkmap paint mode the shape of the brush is ignored, but you can still change the size of the brush (with A and D keys) and its rotation (Q and E keys). Painting is done with the left mouse button.
