Animated objects (actors)

In Panda3D animated meshes are called Actors, the same convention is true for the Koparka Editor.

Placing Actors is identical to placing single objects, with the only difference that actors have animations, refer to the part of the manual dealing with placing single objects for more details.

Currently the editor will only play one idle animation in a loop, but an actor can have any number of animations and these can be played using some custom logic. It’s up to the game designers to make use of custom animations.
