Painting Walls

Fence build with Paint Wall mode

Painting walls, fences and roads from multiple segments can be a slow process and aligning elements can be hard if. The Wall Paint mode is made to help in these situations – after placing the first element of a wall the editor in this mode will automatically align the next element of the wall to fit the previous one, also in this mode, once a object is placed the rotation of future elements is controlled by the mouse cursor – the objects will try to ‘look at’ the mouse cursor.

Paint Walls icon

To place objects on the map, click on the name of the objects on the list visible on the right side of the window and then click on the terrain where you want to place that object. You can press the Tab key to switch to another version of the current object.

The size, orientation and height (z-offset) can be changed before placing the object on the map. You can use the same keys that control the brush in other paint modes, these are:

A and D to change the scale of the object
W and S to change the height of the object
Q and E to change the rotation of the object
1, 2 and 3 to change the current axis of rotation (1 -heading, 2 -pitch, 3 -roll)
Esc to cancel

Grid Snap

It is common that some objects need to be placed in a regular fashion, the grid snap setting helps with that.

The editor will place objects at the nearest multiplication of the grid snap value. For example if the grid snap is 0.5 (the default value) and the mouse pointer is at coordinates x=10.53567, y= 346.789 a object would be placed at x=10.5, y=347.0.

Properties

The editor allows to set custom properties on any object. These properties are simple strings saved with the map files and it’s up to the end user to give them any meaning. The demo.py and demo_util.py scripts may give some idea how properties can be used.

To add properties to an object just enter the text in the properties window before placing the object on the terrain.
