Requirements

To run the editor you will need Panda3D version 1.9 or later. You can get it here:
http://www.panda3d.org/download.php

The editor should also run on recent development builds:
http://buildbot.panda3d.org/downloads/

To use all the features of the editor a GPU (and driver) supporting OpenGL 3.1 is recommended. The program should also run on older versions as long as the GPU/driver supports either GL_ARB_draw_instanced or GL_EXT_draw_instanced extensions. These extensions are used to render grass so disabling grass allows to run the editor on OpenGL 2.1 compatible hardware.

The editor has been developed and runs (at about 30fps) on a machine with the fallowing hardware:

  • CPU: Intel Core2 6300 @ 1.86GHz
  • RAM: 2GB
  • OS: Windows XP 32-bit
  • GPU: ATI Radeon HD 3850 (512MB)
These are the minimum recomended hardware requirements.
