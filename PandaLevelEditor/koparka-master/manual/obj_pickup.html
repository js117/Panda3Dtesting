Picking up objects

Pickup mode icon

As the name suggests the pickup mode allows to pick up already placed objects, move, scale rotate them, edit custom properties or delete the object altogether.

To select a object click on the arrow icon and then click on the object you want to edit.

Once a object is selected you can edit its position (X,Y, Z input fields), rotation (H, P, R input fields) and scale (the Scale input field). If the edited object is a light, the color of that light can also be changed. You can also edit the objects custom properties. All these modifications will be applied to the object after clicking the Apply button.

If you click on the Pick Up button the object can also be moved manually (with the mouse and keys), just like in place object mode.

Clicking the Delete button will permanently remove the object from the map.
