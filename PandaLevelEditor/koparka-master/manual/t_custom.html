Adding custom textures

Custom textures can be added to the editor by placing them in the right directory. Do note that a pair of textures is expected – a diffuse (color) texture and a normal (+gloss in alpha) map- both need to be present.

You will need to place the diffuse texture in the tex/diffuse/ folder and the normal map in the tex/normal/ folder. Both files should have the same name!

The editor wil accept png, tga and dds textures – for best performance it is advised to use dds (dxt) compressed textures.

As noted above the alpha channel of the normal texture is used to map specular highlighs strength (glossines), so it is best to use a format (and editor) that saves the information in the alpha channel (like dds dxt3 or dxt5).
